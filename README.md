# README #

This README documents the steps necessary to get the application up and running.

### What is this repository for? ###

* Markey is a simple app that demonstrates streaming stock market prices via Lightstreamer using a Lightstreamer demo endpoint.

### How do I get set up? ###

* Markey was developed and tested using XCode 16.4 and iOS 18.6. He is a light mode and dark mode supporter.
